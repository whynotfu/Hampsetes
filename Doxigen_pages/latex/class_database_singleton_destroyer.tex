\doxysubsection{Detailed Description}
Класс-\/деструктор для синглтона базы данных 

Класс \doxylink{class_database_singleton_destroyer}{Database\+Singleton\+Destroyer} отвечает за корректное удаление экземпляра синглтона при завершении работы приложения. Гарантирует освобождение ресурсов и закрытие соединения с БД.

Реализует паттерн RAII (Resource Acquisition Is Initialization) для автоматического управления ресурсами.
