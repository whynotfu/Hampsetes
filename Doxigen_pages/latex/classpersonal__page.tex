\doxysubsection{Detailed Description}
Личный кабинет пользователя 

Класс \doxylink{classpersonal__page}{personal\+\_\+page} предоставляет интерфейс личного кабинета для зарегистрированных пользователей казино "{}Хампстерс"{}. Функциональность включает\+:
\begin{DoxyItemize}
\item Отображение информации профиля пользователя
\item Просмотр текущего баланса робуксов
\item Статистика игровой активности (игры, выигрыши, проигрыши)
\item История игровых сессий
\item Достижения и рейтинги (если реализованы)
\end{DoxyItemize}

Личный кабинет позволяет игрокам отслеживать свой прогресс и анализировать игровую статистику. \begin{Desc}
