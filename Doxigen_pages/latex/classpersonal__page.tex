\doxysubsection{Detailed Description}
Класс персональной страницы пользователя 

Класс \doxylink{classpersonal__page}{personal\+\_\+page} реализует интерфейс личного кабинета игрока, где он может просматривать свою игровую статистику и управлять аккаунтом.

Функциональность страницы\+:
\begin{DoxyItemize}
\item Отображение текущего баланса робуксов
\item Статистика по всем играм (количество игр, выигрыши, проигрыши)
\item История последних игровых сессий
\item Общий процент выигрышей
\item Время регистрации и последней активности
\item Настройки профиля пользователя
\end{DoxyItemize}

Страница предоставляет подробную аналитику игровой активности для помощи игроку в отслеживании своих результатов.
