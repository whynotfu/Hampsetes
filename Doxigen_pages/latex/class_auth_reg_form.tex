\doxysubsection{Detailed Description}
Форма авторизации и регистрации пользователей 

Класс \doxylink{class_auth_reg_form}{Auth\+Reg\+Form} представляет диалоговое окно для входа в систему и регистрации новых пользователей. Поддерживает\+:
\begin{DoxyItemize}
\item Авторизацию существующих пользователей
\item Регистрацию новых аккаунтов
\item Переключение между режимами авторизации и регистрации
\item Сетевое взаимодействие с сервером (если необходимо)
\end{DoxyItemize}

Форма является точкой входа в приложение и первым экраном, который видит пользователь при запуске.
