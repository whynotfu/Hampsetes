\doxysubsection{Detailed Description}
Форма авторизации и регистрации пользователей 

Класс \doxylink{class_auth_reg_form}{Auth\+Reg\+Form} предоставляет интерфейс для входа в систему и регистрации новых пользователей. Форма поддерживает\+:
\begin{DoxyItemize}
\item Авторизацию существующих пользователей
\item Регистрацию новых аккаунтов
\item Переключение между режимами входа и регистрации
\item Сетевое взаимодействие с сервером через TCP
\item Валидацию введенных данных
\end{DoxyItemize}

После успешной авторизации пользователь получает доступ к основному меню казино и всем его функциям. \begin{Desc}
