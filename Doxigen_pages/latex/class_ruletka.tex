\doxysubsection{Detailed Description}
Игра "{}Рулетка"{} -\/ ставки на цвета 

Класс \doxylink{class_ruletka}{Ruletka} реализует упрощенную версию классической казино-\/игры "{}Рулетка"{}. Игровая механика включает\+:
\begin{DoxyItemize}
\item Ставки на три цвета\+: красный, черный, зеленый
\item Три размера ставок\+: 100, 500, 1000 робуксов
\item Различные коэффициенты выплат для каждого цвета
\item Анимацию вращения рулетки
\end{DoxyItemize}

Правила игры\+:
\begin{DoxyItemize}
\item Игрок выбирает размер ставки (100/500/1000 робуксов)
\item Игрок выбирает цвет для ставки (красный/черный/зеленый)
\item При нажатии "{}Играть"{} рулетка случайно останавливается на одном из цветов
\item Выплаты\+: красный/черный -\/ 2\+:1, зеленый -\/ 14\+:1 (джекпот) 
\end{DoxyItemize}\begin{Desc}
