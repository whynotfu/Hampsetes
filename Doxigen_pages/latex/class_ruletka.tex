\doxysubsection{Detailed Description}
Игра в рулетку с цветными секторами 

Класс \doxylink{class_ruletka}{Ruletka} реализует классическую игру в рулетку со следующими особенностями\+:
\begin{DoxyItemize}
\item Ставки на красный, черный или зеленый цвет
\item Возможность выбора размера ставки (100, 500, 1000 robux)
\item Анимация вращения рулетки
\item Автоматический подсчет выигрышей
\end{DoxyItemize}

Правила игры\+:
\begin{DoxyItemize}
\item Красный/Черный\+: коэффициент выигрыша x2
\item Зеленый\+: коэффициент выигрыша x14 (редкий сектор)
\item При проигрыше ставка теряется полностью 
\end{DoxyItemize}
