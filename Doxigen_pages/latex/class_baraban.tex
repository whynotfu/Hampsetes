\doxysubsection{Detailed Description}
Игра в слот-\/машину с тремя барабанами 

Класс \doxylink{class_baraban}{Baraban} реализует классическую игру в слоты с следующими особенностями\+:
\begin{DoxyItemize}
\item Три барабана с различными символами
\item Возможность выбора размера ставки (100, 500, 1000 robux)
\item Анимация вращения барабанов
\item Автоматический подсчет выигрышей
\item Различные комбинации для выигрыша
\end{DoxyItemize}

Правила игры\+:
\begin{DoxyItemize}
\item Три одинаковых символа = большой выигрыш
\item Два одинаковых символа = малый выигрыш
\item Размер выигрыша зависит от ставки 
\end{DoxyItemize}
