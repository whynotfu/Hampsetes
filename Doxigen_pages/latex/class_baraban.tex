\doxysubsection{Detailed Description}
Игра "{}Слоты"{} -\/ классические игровые автоматы 

Класс \doxylink{class_baraban}{Baraban} реализует популярную казино-\/игру "{}Слоты"{} с тремя барабанами. Игровая механика включает\+:
\begin{DoxyItemize}
\item Три независимых барабана с различными символами
\item Систему ставок (100, 500, 1000 робуксов)
\item Расчет выигрышей на основе комбинаций символов
\item Анимацию вращения барабанов
\item Интеграцию с балансом пользователя через сервер
\end{DoxyItemize}

Правила игры\+:
\begin{DoxyItemize}
\item Игрок выбирает размер ставки
\item При нажатии "{}Играть"{} все три барабана случайно останавливаются
\item Выигрыш определяется по количеству одинаковых символов в ряд
\item Джекпот -\/ три одинаковых символа (умножение ставки на коэффициент) 
\end{DoxyItemize}\begin{Desc}
