\doxysubsection{Detailed Description}
Класс \doxylink{class_baraban}{Baraban}. 

Класс \doxylink{class_baraban}{Baraban} реализует окно игрового автомата, позволяющее пользователю устанавливать ставки, запускать игровой процесс (слоты) и управлять своим балансом. Также предоставляет возможность возвращения в главное меню.
