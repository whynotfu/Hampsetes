\doxysubsection{Detailed Description}
TCP-\/сервер для обработки клиентских подключений казино 

Класс \doxylink{class_my_tcp_server}{My\+Tcp\+Server} реализует серверную логику казино "{}Хампстерс"{}. Основные функции\+:
\begin{DoxyItemize}
\item Обработка множественных TCP-\/подключений клиентов
\item Парсинг и маршрутизация клиентских запросов
\item Взаимодействие с базой данных через \doxylink{class_data_base_singleton}{Data\+Base\+Singleton}
\item Управление сессиями пользователей
\item Обработка игровых операций и административных команд
\end{DoxyItemize}

Сервер поддерживает следующие типы запросов\+:
\begin{DoxyItemize}
\item AUTH\+: авторизация пользователей
\item REG\+: регистрация новых пользователей
\item GAME\+: игровые операции и обновление балансов
\item ADMIN\+: административные функции
\item STATS\+: получение статистики пользователей
\end{DoxyItemize}

Протокол связи основан на текстовых командах, передаваемых через TCP.
