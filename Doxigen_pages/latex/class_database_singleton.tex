\doxysubsection{Detailed Description}
Синглтон для управления базой данных казино 

Класс \doxylink{class_database_singleton}{Database\+Singleton} реализует паттерн Singleton для обеспечения единственного экземпляра подключения к базе данных в течение всего времени работы приложения.

Функциональность\+:
\begin{DoxyItemize}
\item Авторизация и регистрация пользователей
\item Управление балансами игроков
\item Получение статистики пользователей
\item Проверка прав доступа (admin/user)
\item Ведение истории игр и выигрышей
\end{DoxyItemize}

Паттерн Singleton гарантирует\+:
\begin{DoxyItemize}
\item Единственное подключение к БД
\item Централизованное управление данными
\item Эффективное использование ресурсов 
\end{DoxyItemize}
