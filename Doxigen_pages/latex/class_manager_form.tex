\doxysubsection{Detailed Description}
Главный менеджер форм приложения казино 

Класс \doxylink{class_manager_form}{Manager\+Form} является центральным контроллером, который управляет всеми окнами и формами приложения. Он отвечает за\+:
\begin{DoxyItemize}
\item Инициализацию всех игровых окон
\item Установку связей между формами через сигналы и слоты
\item Координацию переходов между различными разделами приложения
\end{DoxyItemize}

Архитектура основана на паттерне "{}Медиатор"{}, где \doxylink{class_manager_form}{Manager\+Form} выступает посредником между всеми формами приложения.
