\doxysubsection{Detailed Description}
Класс игры-\/кликера для заработка робуксов 

Класс \doxylink{class_kliker}{Kliker} реализует простую игру-\/кликер, где игрок может заработать робуксы многократным нажатием на кнопку.

Механика игры\+:
\begin{DoxyItemize}
\item Игрок нажимает на кнопку для получения робуксов
\item За каждое нажатие начисляется определенное количество робуксов
\item Заработанные робуксы автоматически добавляются к балансу игрока
\item Игра предназначена для пополнения баланса без риска потерь
\end{DoxyItemize}

Это альтернативный способ получения игровой валюты помимо выигрышей в азартных играх казино.
