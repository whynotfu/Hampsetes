\doxysubsection{Detailed Description}
Главное окно казино с меню выбора игр 

Класс \doxylink{class_casino_main_window}{Casino\+Main\+Window} представляет центральное меню приложения казино. Предоставляет пользователю возможность\+:
\begin{DoxyItemize}
\item Выбирать из доступных мини-\/игр
\item Просматривать превью игр с изображениями
\item Переходить к админ-\/панели (для администраторов)
\item Открывать личную страницу пользователя
\item Просматривать текущий баланс и статистику
\end{DoxyItemize}

Окно служит хабом для навигации по всем разделам приложения.
