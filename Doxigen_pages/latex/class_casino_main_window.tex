\doxysubsection{Detailed Description}
Главное окно казино с интерфейсом выбора мини-\/игр 

Класс \doxylink{class_casino_main_window}{Casino\+Main\+Window} представляет основное меню казино "{}Хампстерс"{}. Предоставляет пользователю возможность\+:
\begin{DoxyItemize}
\item Просматривать доступные мини-\/игры с помощью карусели
\item Запускать выбранную игру
\item Переходить в админ-\/панель (для администраторов)
\item Переходить в личный кабинет
\item Возвращаться к форме авторизации
\end{DoxyItemize}

Окно отображает визуальные превью игр и их названия, позволяя пользователю переключаться между различными развлечениями. \begin{Desc}
\item[Examples]\par
