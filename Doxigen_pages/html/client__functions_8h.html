Detailed Description

Заголовочный файл вспомогательных функций для казино.

Этот файл содержит объявления функций для авторизации, регистрации, получения пути к ресурсам, генерации случайного числа для игровых слотов и перемешивания массива.

Генерирует случайное число для игровых слотов.

Функция генерирует три случайных числа в диапазоне от 1 до 3. Если все три числа равны, возвращается это число (1, 2 или 3); в противном случае возвращается 0.

Returns
Случайное число от 1 до 3, если все числа совпадают, иначе 0.
