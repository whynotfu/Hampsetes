Detailed Description

Личный кабинет пользователя

Класс personal_page предоставляет интерфейс личного кабинета для зарегистрированных пользователей казино "Хампстерс". Функциональность включает:

  • Отображение информации профиля пользователя
  • Просмотр текущего баланса робуксов
  • Статистика игровой активности (игры, выигрыши, проигрыши)
  • История игровых сессий
  • Достижения и рейтинги (если реализованы)

Личный кабинет позволяет игрокам отслеживать свой прогресс и анализировать игровую статистику.
