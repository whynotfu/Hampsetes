Detailed Description

Реализация вспомогательных функций для казино.

Этот файл содержит реализации функций для авторизации, регистрации, получения пути к ресурсам, генерации случайного числа для игровых слотов и перемешивания массива.

Функция служит заглушкой для проверки авторизации пользователя. Возвращает true, если логин равен "bobr" и пароль равен "123", иначе — false.

Проверяет корректность данных для регистрации.

Функция служит заглушкой для регистрации пользователя. Если одно из полей (логин, пароль или проверка пароля) пустое, возвращается false. Если логин не равен "bobr" и пароль совпадает с подтверждением, возвращается true.

Parameters

Returns
true, если регистрация может быть выполнена, иначе false.

◆ slots_rand()

int slots_rand ( )

Генерирует случайное число для игровых слотов.

Функция генерирует три случайных числа в диапазоне от 1 до 3. Если все три числа равны, возвращается это число (1, 2 или 3). Если хотя бы одно число отличается от других, возвращается 0.

Returns
Случайное число от 1 до 3, если все числа совпадают, иначе 0.
